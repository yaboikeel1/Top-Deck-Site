# Contributing to Top Deck Music Group

## Before making changes

- Confirm you are working from `develop` or a feature branch.
- Do not commit passwords, API keys, Stripe secrets, or private information.
- Keep website content in `public/data/site-data.json` when possible.

## Before merging into `main`

- Open every affected page on mobile and desktop.
- Test navigation links.
- Test Netlify forms.
- Confirm images and audio load correctly.
- Confirm no secret keys are present in the repository.
- Use a clear commit message describing the change.
